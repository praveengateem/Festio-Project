# 🎉 Festio - Event Management App

## 📌 Description
Festio is a web application that allows users to explore events, book tickets, and manage their profiles easily.

## 🚀 Features
- 🔐 User Authentication (Login & Register)
- 📅 Event Listing
- 🎟️ Ticket Booking
- 📊 Dashboard
- 👤 Profile Management

## 🛠️ Tech Stack
- React.js
- Vite
- Tailwind CSS
